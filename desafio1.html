Tag img

A tag img é usada para inserir imagens na página. Ela é uma tag vazia, ou seja, ela não tem conteúdo nem fecha com uma barra (/). Ela tem dois atributos obrigatórios: src e alt. O atributo src indica o endereço da imagem, que pode ser uma URL ou um caminho relativo. O atributo alt indica um texto alternativo que será exibido caso a imagem não possa ser carregada. Por exemplo:

Esta é a imagem do logo da empresa.

Tags ul e li

As tags ul e li são usadas para criar listas não ordenadas na página. A tag ul é a tag que contém a lista, e a tag li é a tag que contém cada item da lista. As tags ul e li são tags de conteúdo, ou seja, elas podem conter texto e outras tags dentro delas. Por exemplo:

  • Item 1
  • Item 2
  • Item 3

Esta é uma lista não ordenada com três itens.

Tag table

A tag table é usada para criar tabelas na página. Ela é uma tag de conteúdo, ou seja, ela pode conter texto e outras tags dentro dela. A tag table é composta por outras tags que definem a estrutura da tabela, como thead, tbody, tr, th e td. A tag thead indica o cabeçalho da tabela, a tag tbody indica o corpo da tabela, a tag tr indica uma linha da tabela, a tag th indica uma célula de cabeçalho da tabela, e a tag td indica uma célula de dados da tabela. Por exemplo:

Nome Idade Profissão
Ana 25 Engenheira
Bruno 32 Médico
Carla 28 Professora

Esta é uma tabela com três linhas e três colunas.

Tag form

A tag form é usada para criar formulários na página. Ela é uma tag de conteúdo, ou seja, ela pode conter texto e outras tags dentro dela. A tag form tem dois atributos importantes: action e method. O atributo action indica o endereço para onde os dados do formulário serão enviados, e o atributo method indica o método de envio dos dados, que pode ser GET ou POST. Dentro da tag form, podemos usar outras tags para criar os campos do formulário, como input, label, select, option, textarea e button. Por exemplo:
